###Requirements

* node
* Sketch.app

###Installation

1. Install sketch-tool `$ curl -L https://raw.githubusercontent.com/cognitom/dotfiles/master/lib/sketchtool.sh | sudo sh`
2. Run `$ npm install`

###Usage

1. Run `$ gulp`
2. Work on and save either `app.coffee` or `assets.sketch` in the `src` folder.
